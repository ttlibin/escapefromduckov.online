Installation Checklist

  1. Subscribe via Steam Workshop.
  2. Verify load order inside Escape From Duckov's mod manager tab.
  3. Back up save files at AppData\LocalLow\Duckov.
  4. Launch game with "Enable Mods" toggled on splash screen.

Compatibility Notes

Escape From Duckov is officially solo, but community discovered a stable two-player method:

  • Create a private network with Radmin VPN.
  • Install Multiplayer Package mod bundle.
  • Set both clients to "Host Migration" mode before launching a raid.
  • Use in-game console to sync inventories after extraction.

Disclaimer: expect desyncs; always back up saves.
