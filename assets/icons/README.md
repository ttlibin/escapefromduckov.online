# Escape From Duckov Website Icons

## Icon Concepts

### 1. Primary Logo (duckov-logo-main.svg)
- **Style**: Minimalist military duck icon
- **Features**: Clean geometric design, game controller elements
- **Colors**: Dark theme with gold accent (#f2c046)

### 2. Favicon (duckov-favicon.ico)
- **Style**: Compact duck silhouette
- **Features**: Simple, recognizable at small sizes
- **Background**: Transparent

### 3. App Icons (duckov-app-icon.png)
- **Style**: Tactical duck with extraction theme
- **Features**: Modern flat design, suitable for app stores
- **Sizes**: 512x512px, 256x256px, 128x128px, 64x64px

### 4. Social Media (duckov-social.png)
- **Style**: Horizontal banner format
- **Features**: Duck character with game elements
- **Sizes**: 1200x630px (Twitter header), 1200x600px (Facebook)

### 5. Discord Avatar (duckov-discord.png)
- **Style**: Circular duck portrait
- **Features**: Discord-friendly square format
- **Sizes**: 512x512px, 128x128px

### 6. Steam Capsule (duckov-steam.jpg)
- **Style**: Vertical capsule format
- **Features**: Duck with military equipment
- **Sizes**: 460x215px (Steam capsule)

### 7. Mobile Icon (duckov-mobile.png)
- **Style**: Simplified duck outline
- **Features**: High contrast, touch-friendly
- **Sizes**: 192x192px, 144x144px

## Design Principles

- **Color Palette**:
  - Primary: #0f141a (dark duck blue)
  - Secondary: #f2c046 (gold/accent)
  - Dark: #080c10 (background)
  - Light: #e5ecf5 (text)

- **Typography**: Modern sans-serif (Inter font family)
- **Visual Style**: Tactical gaming aesthetic
- **Iconography**: Game-related symbols (extract, ammo, duck)
